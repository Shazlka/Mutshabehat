Release Note — V72 Mobile Details + HD Share

Implemented:
- Refresh now returns to the last opened screen/database instead of always going to Home.
- Surah filter panel is collapsed by default on first open and when switching database.
- Added swipe-down-to-close support for modals on iPhone/iPad/mobile layouts.
- On iPhone/iPad/mobile, tapping a group opens details in a separate window instead of expanding inside the list.
- Copy group now includes title, group number, surahs, ayat, notes, and additional/unique notes.
- Added HD PNG image export/share button for each group, preserving color-coded ayah parts and notes.
- Surah tags in the group header are highlighted light red when that Surah contains a unique color-coded part.
- Favorite/completed text badges beside the title were removed; only icons/colors change.
- Removed the completed icon from the group action buttons.
- The group number now toggles completed status; completed group number turns light green.
- Added new color coding type: diff3 with light teal/cyan style.

Preserved:
- V71 Surah/Ayah dependent dropdowns in Add/Edit windows.
- Settings font preview.
- Personal and Automated database separation.
- Smart Quran search and existing filters.
