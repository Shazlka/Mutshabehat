## Release Note — V38

Source files used: Mutashabihat_v38.zip, containing index.html, styles.css, app.js, and release note.txt.

### V38 Restore Scope
- Restored all features reported as removed after version 34.
- This version keeps the V37 edit-search improvements and adds a complete V38 restore layer for Add Group, Modification/Edit Group, group cards, and management tools.

### Add Group Window
- Restored smart Quran search from `quran-reference.js`.
- Added search bar in Add New Group.
- Search ignores tashkeel, hamza differences, and common Arabic letter variations.
- Search results are ordered with exact matches first and normalized/non-exact matches after.
- Each search result includes checkbox, Surah name, Ayah number, ayah text, Add full ayah, and Add selected text.
- Selecting a checkbox automatically adds the ayah to the draft.
- Deselecting a checkbox automatically removes the ayah from the draft.
- Restored live ayah preview.
- Restored group color picker and color preview.
- Restored auto-generate group title.

### Modification / Edit Group Window
- Restored smart Quran search from `quran-reference.js`.
- Added search bar in Modification/Edit Group.
- Search ignores tashkeel, hamza differences, and common Arabic letter variations.
- Search results are ordered with exact matches first and normalized/non-exact matches after.
- Each search result includes checkbox, Surah name, Ayah number, ayah text, Add full ayah, and Add selected text.
- Selecting a checkbox automatically adds the ayah to the group being edited.
- Deselecting a checkbox automatically removes the ayah from the group being edited.
- Restored live ayah preview in Modification window.
- Restored group color picker and color preview.
- Restored auto-generate group title.

### Group Card Window
- Restored improved lock icon with locked/unlocked status.
- Locked groups cannot be saved, deleted, or merged until unlocked.
- Restored visual inline compare function inside the group body.
- Compare highlights shared/same words and different words with different colors.

### Management Tools
- Restored floating/draggable behavior for all modal windows.
- Restored Merge Groups window.
- Restored Duplicate Detection window.

# Release Note — V37

Source files used: Mutashabihat_v37.zip, containing index.html, styles.css, app.js, and release note.txt.

## V37 New Features

### Modification / Edit Group Window
- Improved Quran search header in the Modification window.
- While typing a search word, the search header now shows:
  - Exact match count.
  - Close / normalized match count.
  - Total result count.
- Search result ayah text now highlights the searched word to make it easier to see.
- Exact matches are highlighted with a yellow mark.
- Close / normalized matches are highlighted with an orange mark.
- Close matching continues to ignore:
  - tashkeel
  - hamza differences
  - common Arabic letter variations

### Group Card Window
- Removed the small list / handle icon from the group action area.
- The removed icon is the small stacked-line handle shown beside the edit/check/star/compare/lock icons.

### Mobile Group Card Layout
- Maintained V36 mobile behavior where group action icons appear below the group title.
- Desktop / large screen layout remains unchanged except that the old handle icon is hidden.

# 18. Important File Structure Note

V35 package contains:

*   `index.html`
*   `styles.css`
*   `app.js`

But the app still depends on external project files:

*   `data.js`
*   `quran-reference.js`
*   optional `fonts/` folder for custom fonts

These are referenced by `index.html` and must remain in the same folder when running the app.
