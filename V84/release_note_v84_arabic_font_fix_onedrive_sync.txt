V84 - Arabic Font Fix & OneDrive Sync
=====================================
Release Date: 2026-05-16

Changes in V84:
- Fixed Arabic font rendering in all app windows (settings, modals, buttons)
  Root cause: Readex Pro, Noto Naskh Arabic, and Cairo fonts were not loaded
  from Google Fonts CDN. Added proper preconnect and stylesheet links to index.html.
- Added OneDrive Sync feature: backup personal database to OneDrive automatically
  alongside GitHub sync. Uses Microsoft Graph API with PKCE OAuth (no client secret).
  Azure App Registration: Mutshabehat OneDrive Sync (client ID: 16371ab6-...)
- App version bumped to V84 with updated cache-busting version strings.

Bug Analysis (V82 vs V83 vs V84):
- V82 app.js: 103,500 chars - included extra debugging/legacy code
- V83 app.js: 86,546 chars - refactored, cleaner codebase
- V84 app.js: same as V83 + OneDrive sync addition (~7,750 chars)
- Font issue existed since V70 but only visible on systems without those fonts installed

To Connect OneDrive (first time):
1. Open app Settings
2. Scroll to OneDrive Sync section
3. Click Connect OneDrive - sign in with Microsoft account
4. Enable Auto-sync checkbox and Save
